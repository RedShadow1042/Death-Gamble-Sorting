# Death Gamble Sorting 🎲💀

A chaotic twist on the classic Bubble Sort algorithm.

## How it Works
This isn't your average sorting algorithm. It functions like a standard **Bubble Sort**, but with a dangerous twist:
* After every successful swap/step, the algorithm generates a "Death Number."
* If the gamble fails (the death number hits), the entire array is immediately shuffled using a **Bogo Sort** style randomization.
* The sort then has to start all over again from the beginning.
